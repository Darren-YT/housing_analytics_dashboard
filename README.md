# Housing Analytics Dashboard

Interactive dashboard built with Streamlit and Python for analyzing housing affordability, environmental risks, and demographic trends across U.S. states.

## Features
- Housing price and income analysis
- PM2.5 environmental data integration
- Crime and disaster data visualization
- Interactive state comparison dashboard
- Data cleaning and visualization workflows

## Technologies
- Python
- Streamlit
- Pandas
- Census API
- EPA API
- FEMA Data
